# Silk Markdown

一个丝滑的 VS Code Markdown 阅读器 — 精美预览、语法高亮、大纲导航，一切尽在指尖。

## 功能特性

- **沉浸式阅读模式** — 打开 `.md` 文件自动进入阅读模式，排版精美
- **One Dark Pro 语法高亮** — 代码块按语言差异化配色
- **右侧大纲导航** — 固定目录面板，自动跟踪当前阅读位置
- **页内链接跳转** — 点击 Markdown 链接在阅读模式内无缝导航
- **代码块一键复制** — 悬停代码块即可复制
- **页内搜索** — `Ctrl+F` 在预览中搜索内容
- **阅读/编辑切换** — 标题栏按钮或 `Ctrl+Shift+V` 一键切换

## 使用方法

1. 打开任意 `.md` 文件，自动进入阅读模式
2. 点击标题栏**编辑图标**（或按 `Ctrl+Shift+V`）切换到源码编辑
3. 再次点击**阅读图标**（或按 `Ctrl+Shift+V`）切回阅读模式
4. 右键 `.md` 文件 → "打开方式" 可选择阅读模式或文本编辑器
